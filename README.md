# QA Automation Portfolio - Playwright

## 📌 Project Overview
End-to-end test automation framework built with Playwright using Page Object Model and data-driven testing.

## 📂 Project Structure
/tests → test cases  
/pages → page objects  
/data → test data (users & products)

## 🚀 Features
- E2E purchase flow automation
- Page Object Model (POM)
- Data-driven testing
- Modular and scalable structure

## 🧪 Test Scenarios
- Add product to cart
- Validate cart information
- Complete checkout process
- Validate successful purchase

## 🛠 Tech Stack
- Playwright
- JavaScript (Node.js)

## ▶️ Run Tests
```bash
npx playwright test
